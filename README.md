# Finance Tracker Frontend 
Basic Frontend display for finance tracker. Four views: summary, transactions, accounts, and categories. 

## Live Demo
https://finance-tracker-frontend-neon.vercel.app/ 

## Tech Stack 
- **React** - Frontend framework 
- **Tailwind CSS** - Styling
- **Axios** - API calls
- **React Router** - Navigation between views 

## Features
- Summary View: Shows user's income, expense, and net balance across all accounts
- Transactions: Add and delete transactions, differs between income and expense, select date, account, category
- Accounts: Add and delete account with income and expense breakdown for each account 
- Category: Add and delete category with associated transactions listed under each category

## Getting Started 
**1. Clone the repo** 
```bash 
git clone https://github.com/ahuynh6265/finance-tracker-frontend.git 
cd finance-tracker-frontend
```

**2. Install dependencies**
```bash
npm install
```

**3. Start server**
```bash
npm start
```

> Backend is hardcoded in api.js for the demo
